# Sistema de Controle de Estoque - Escola de Enfermagem

Projeto desenvolvido para a disciplina de Front-end (2º Semestre/3º Semestre) da faculdade **Unicesumar Londrina**.

### 👨‍💻 Autor

* **Aluno:** José Flávio Xavier
* **Orientador:** Prof. Leonardo Rocha

### 📝 Resumo do Sistema

Sistema voltado para o controle de estoque de uma escola de enfermagem, visando solucionar a falta de agilidade no monitoramento, lançamento e baixa de insumos utilizados em atividades acadêmicas.

### 🛠 Tecnologias Utilizadas

* **Front-end:** HTML5, CSS3, JavaScript (ES6+).
* **API:** MockApi (Simulação de banco de dados e persistência de dados).

### ⚙️ Funcionalidades

* Listagem de materiais cadastrados.
* Validação de retirada (Lógica de segurança de estoque).
* Atualização de saldo (Baixa via método PUT).
* Remoção de itens (DELETE).

### 🚀 Como executar

1. Faça o download ou clone este repositório.
2. Abra a pasta do projeto em seu editor de código (VS Code, etc).
3. Abra o arquivo `index.html` diretamente em seu navegador.
4. Utilize o sistema conforme as opções de cadastro e retirada.
